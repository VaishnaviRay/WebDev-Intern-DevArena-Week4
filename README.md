# Lumina - E-commerce Product Landing Page

A premium, fully responsive e-commerce landing page designed for "Lumina", a fictional high-end audio equipment brand. This project demonstrates modern web design principles, including glassmorphism, gradient accents, and mobile-first responsiveness using pure HTML, CSS, and JavaScript.
